# Iteration 7 — Voice Script Catalog (사전 녹음 음성 스크립트)

**문서 버전**: 1.0
**작성일**: 2026-04-29
**상위 문서**: `iteration7-requirements-patch.md` §FR-8.4.3
**대상 청취자**: 외부 녹음 작업자 / 발주자
**한국어 톤**: 근엄·차분·고전적 진행자 (1인 발화자 권장)

---

## 1. 목적

본 문서는 마피아 게임 진행 안내 음성을 사전 녹음하기 위한 **단일 명세**입니다. 다음을 정의합니다:

1. **녹음 대상 시점 27건** 의 안정 식별자 (`audioId`)
2. 각 시점의 **음성 텍스트** (정형 멘트 — 변수 제거)
3. 각 시점의 **자막 텍스트** (변수 보간 형식, 참고용 — 코드가 자동 생성)
4. 발화 가이드라인 (톤·길이·강세)

> **개발자 안내**: 본 문서의 `audioId` 그대로 `web/public/audio/<audioId>.mp3` 파일명으로 저장하면 자동으로 매핑됩니다. 누락된 파일은 콘솔 경고만 남기고 자막만 표시됩니다(graceful skip).

---

## 2. 정형화 정책 (요약)

- **자막**: 동적 변수(이름·일수·시간·역할) 그대로 보간 — 코드 카탈로그가 생성.
- **음성**: 변수 부분을 일반화 표현으로 대체. 정체 분기처럼 의미가 갈리는 경우만 **2벌 이상 분리**.
- **변수 일반화 규칙**:
  - `%s` (이름) → "한 사람" 또는 "다음 차례입니다" 등 문맥에 맞게 일반화 (음성에서 이름 제거)
  - `%d` (일수) → "새로운 아침이"
  - `%d초` (인트로 시간) → 시간 어구 제거
  - `%s` (역할 — Eliminated) → 마피아 vs 비마피아 2벌 (사용자 가이드 적용)

## 4. 발화 가이드라인 (외부 녹음 작업자 안내)

### 4.1 톤·캐릭터
- **인물**: 근엄한 노년 진행자 또는 차분한 중저음 남성 (남성 한정 권장 — 일관성)
- **분위기**: 1940~50년대 라디오 드라마 진행자 (Iteration 6 노이르 디자인과 일관)
- **속도**: 보통보다 약간 느리게 (rate ≈ 0.9). 단어 사이 호흡을 명확히.
- **피치**: 낮은 톤 (pitch ≈ 0.85~0.9 상응)

### 4.2 길이·포맷
| 항목 | 권장 |
|---|---|
| 파일 포맷 | MP3, 44.1 kHz, 96~128 kbps |
| 채널 | 모노(권장) 또는 스테레오 |
| 1 파일 길이 | 2~12초 |
| 1 파일 크기 | < 200 KB |
| 시작 무음 | 0.1~0.2초 권장 (재생 클릭음 방지) |
| 끝 무음 | 0.3~0.5초 권장 (자연스러운 종결) |

### 4.3 시점별 강세 가이드

| audioId | 분위기 |
|---|---|
| `game.started` | 무겁게 선언, 마지막 "받아들이시오" 단호 |
| `phase.night` | 깊고 낮게, "눈을 감으시오" 천천히 |
| `phase.day.first` | 약간 밝지만 여전히 진중하게 |
| `phase.day` | 평이하게 — 매일 반복 사용 |
| `night.mafia` / `night.police` / `night.doctor` | 차분하게, 직책마다 미세하게 다르게(역할에 맞춤) |
| `intro.speaker` | 부드럽게, 무대를 넘기는 톤 |
| `death.announced` | 슬픔과 무게감, "마을의 슬픔" 약간 강조 |
| `peaceful.night` | 안도감 살짝, "아무도" 명확히 |
| `eliminated.mafia` / `eliminated.notmafia` | 후반 정체 공개 부분 강조. 마피아 = 단호 / 비마피아 = 무거운 슬픔 |
| `timer.30` / `timer.10` / `timer.0` | 차분하지만 명료하게, `timer.10` 의 "마음을 정하시오"는 약간 단호 |
| `vote.recount` | 의심·재고 |
| `vote.noelim` | 무거운 안도 |
| `end.mafia` / `end.citizen` | 절정. `end.mafia` = 어두운 승리 / `end.citizen` = 정의의 승리 |
| `end.force` | 평이하게 — 메타 메시지 |
| `game.paused` / `game.resumed` | 차분, 일시적 중단 표현 |
| `voice.on` / `voice.off` | 평이하게 — 짧은 메타 |
| `system.restore` / `system.persist.failure` | 짧고 명료. 토스트 |

### 4.4 파일 명명 규칙
- **파일명**: `<audioId>.mp3` (점·소문자 그대로)
- **저장 위치**: `web/public/audio/`
- 예: `web/public/audio/game.started.mp3`, `web/public/audio/eliminated.mafia.mp3`

> 파일명에 점(`.`)이 포함되어도 정적 파일 서빙·HTTP 라우팅에 영향 없음. 빌드 후 `dist/audio/...` 로 그대로 복사됨.

## 5. 녹음 누락·교체 절차

1. **누락 시점이 발생** — 자막은 표시되고 음성만 빠짐. 콘솔 경고. 게임 진행 무중단.
2. **나중에 mp3 추가** — 파일을 `web/public/audio/<audioId>.mp3` 에 두고 vite/dev 서버 재시작 또는 `npm run build` 재빌드. 별도 코드 변경 불필요.
3. **문구 교정** — 본 문서 §3 표의 "음성 텍스트" 수정 후 mp3 재녹음. audioId 가 같으면 코드 변경 없음.
